# Coursera_PythonforDataScience_Final Assignment
Python Project for Data Science (IBM Data Analyst Professional Certificate) – Stock Data Analysis and Visualization

**Short Description of the Project**
This project analyzes historical stock and revenue data for Tesla and GameStop. It demonstrates an end-to-end workflow including data collection, cleaning, and visualization. The project involves:

Collecting stock data using the yfinance library.
Web scraping quarterly revenue data from HTML pages using BeautifulSoup.
Cleaning and transforming the data to handle missing values, remove unwanted characters, and convert data types for analysis.
Visualizing stock prices and revenue over time using Matplotlib and custom graphing functions to provide insights into company performance.

**Skills Demonstrated**

Python – coding and scripting to perform data collection, cleaning, and analysis.
Pandas – handling DataFrames, merging datasets, and data manipulation.
Requests – downloading HTML pages for web scraping.
Yahoo Finance / yfinance – retrieving historical stock data.
BeautifulSoup – parsing HTML and extracting revenue tables.
Matplotlib – creating visualizations for stock price and revenue trends.
Data Cleaning – dealing with missing values, string formatting, and type conversions.
Data Visualization – creating informative plots to interpret financial trends.
